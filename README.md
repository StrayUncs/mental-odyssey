# Mental Odyssey

This project is for HackSheffield10

We are aiming to create a mental health app to help groups of people through
their mental health journey.

## How to use
First put in your .env file your OPENAI_API_KEY inside
`python -m venv venv`
`source venv/bin/activate  `
`pip install -U langchain`
`pip install -U langchain-openai`
`pip install -U langgraph-supervisor`
`pip install -U langgraph`
